Plugin for automated data migration from IP.Board to bbPress. Convert IP.Board to bbPress easily and fast.


== Description ==

Transfer data from IP.Board to bbPress with CMS2CMS plugin. Save your time with automated forum data migration.


= HOW THIS PLUGIN HELPS: =

Possibility to preview the future results via running the Free Demo Migration;
The approximate time of migration is 15 minutes;
Ability to indicate the content items you want to move from IP.Board to WordPress CMS (Data “Mapping” option);
Availability of live help. If you have any migration related questions, our tech team will try to answer them;
The pricing depends on the amount of content you’re going to move;
Availability of live help. If you have any migration related questions, our tech team will try to answer them;
If you want to perform a customized migration, make a request for a CMS2CMS data migration service package.

= WHAT WILL BE MOVED FROM IP.Board TO bbPress? =

* Forums/Categories;
* Forum Posts;
* Forum Topics;
* Forum Users;
* Forum Images;
* Forum Attachments;
* Forum Topic Watchers;

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the  “CMS2CMS:  Automated IP.Board To bbPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing forum URL address.
7. Download the Connection Bridge and place it into the Existing forum root folder. You can find a detailed instruction here.
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
